Sobre o Projeto

O projeto consiste em criar um site acessível, aplicando os conhecimentos de acessibilidade web. O objetivo é garantir uso por pessoas com diferentes necessidades e habilidades, seguindo boas práticas.

Recursos de Acessibilidade Implementados

  • Estrutura semântica: <header>, <main>, <section>, <footer>.
  • Link “Pular para o conteúdo”.
  • Contraste suficiente e foco visível.
  • Labels associados a campos de formulário.
  • Controles para alto contraste e aumento de fonte.
